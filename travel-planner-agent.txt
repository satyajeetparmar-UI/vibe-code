PROJECT NAME: AI Travel Planner (No Login)

GOAL:
Build a complete React + Vite web application that allows users to enter a destination city and get:
1. Current weather
2. 5-day forecast
3. Nearby tourist attractions
4. Country details
5. A simple AI-generated 1-day travel plan (rule-based, no paid AI API)

No authentication system required.

TECH STACK:
- React (Vite)
- Tailwind CSS
- Axios
- Public APIs only
- Environment variables for API keys
- Clean folder structure
- Fully responsive UI
- Modern card-based layout

FEATURE DETAILS:

1️⃣ Search Functionality:
- User enters city name
- On submit:
  - Fetch coordinates from OpenWeather Geocoding API
  - Fetch current weather
  - Fetch 5-day forecast
  - Fetch country info
  - Fetch nearby tourist attractions

2️⃣ Weather API:
Use OpenWeatherMap free tier:
- Geocoding API
- Current Weather API
- 5-day Forecast API

Store API key in .env:
VITE_WEATHER_API_KEY=your_api_key

3️⃣ Tourist Attractions:
Use Foursquare Places API (free tier)
OR fallback:
Use Wikipedia API search for "Top tourist attractions in {city}"

4️⃣ Country Info:
Use REST Countries API:
https://restcountries.com/v3.1/name/{country}

Display:
- Flag
- Capital
- Currency
- Population

5️⃣ AI Travel Plan (No Paid AI):
Create utility function generatePlan.js
Logic:
- Based on weather condition (Rain, Clear, Clouds)
- Based on number of attractions
- Return formatted 1-day itinerary:
  Morning
  Afternoon
  Evening

Example Output:
Morning: Visit {Top Attraction}
Afternoon: Explore local cafes
Evening: Sunset viewpoint visit
